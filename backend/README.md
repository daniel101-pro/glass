# Glass Backend

A minimal Express + TypeScript API server.

## Scripts
- dev: `npm run dev`
- build: `npm run build`
- start: `npm run start`

## Environment
Create a `.env` file:

```
PORT=4000
NODE_ENV=development
# CORS_ORIGIN=http://localhost:3000
```

## Endpoints
- `GET /` basic info
- `GET /api/v1/health` health check
